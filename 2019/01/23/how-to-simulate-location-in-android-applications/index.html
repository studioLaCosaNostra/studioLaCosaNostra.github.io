How to simulate location in android applications

2019-01-23
android, developer tools, fake gps

To be able to do this, you must activate the developer mode in the android system.

Steps to activate developer mode can be found in google

Steps to activate mock location:

  1. Install the Fake GPS location application.

  2. Go to Developer options

  3. Select mock location app and select application Fake GPS location
